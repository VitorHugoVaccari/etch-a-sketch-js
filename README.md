# Etch a Sketch 

Projeto desenvolvido para praticar manipulação do DOM e eventos com JavaScript, inspirado no clássico brinquedo Etch A Sketch.

## Funcionalidades

* Criação dinâmica de grade com base na entrada do usuário
* Interação com hover (efeito de desenho)
* Geração de cores aleatórias (RGB)
* Escurecimento progressivo a cada interação
* Reset da grade sem recriar os elementos
* Limite de tamanho da grade (1 a 100)

## Tecnologias utilizadas

* HTML5
* CSS3 (Flexbox)
* JavaScript (DOM e eventos)

## O que aprendi

* Manipulação dinâmica de elementos com JavaScript
* Uso de event listeners
* Controle de estado em elementos interativos
* Validação de entrada do usuário
* Diferença entre soluções simples e escaláveis

## Objetivo

Este projeto foi desenvolvido como parte dos estudos no The Odin Project, com foco em consolidar conceitos fundamentais de JavaScript.

## Como usar

1. Insira a quantidade de quadrados desejada (entre 1 e 100)
2. Clique em "Confirmar"
3. Passe o mouse sobre os quadrados para desenhar
4. Use o botão "Reiniciar" para limpar a grade

## Observações

* O projeto foi desenvolvido com foco em aprendizado, priorizando simplicidade e clareza do código
* Algumas decisões foram tomadas considerando o tamanho do projeto, mantendo o código funcional e direto
